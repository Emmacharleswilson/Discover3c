What is 3c Hair?

3C hair is a curly hair type consisting of tight coils with volume, with lots of strands packed together to create texture. 3C was created and added to the hair type chart when it was clear that there was a missing type between 3B and 4A.

In general, Type 3 curly hair is defined by having an “S” shaped curl pattern. Additionally, because the cuticle doesn’t lay as flat as Type 1 or Type 2 hair, the hair also isn’t as shiny.
